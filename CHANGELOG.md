# 更新日志

## [1.1.0] - 2025-12-29

### 新增功能

- 支持 EPUB 格式输出
- 新增 `--format` 选项，支持 pdf/epub/both 三种模式
- EPUB 格式特性：
  - 标准 EPUB 3 格式
  - 完整的目录导航
  - 响应式排版，适合各种阅读器
  - 自动处理图片和代码块样式
  - 文件体积通常小于 PDF

### 技术改进

- 添加 `epub-gen-memory` 依赖用于 EPUB 生成
- 实现文章 HTML 内容提取功能
- 实现并发内容提取机制
- 优化 EPUB 样式，提升阅读体验

### 文档更新

- 更新 README.md，添加 EPUB 使用说明
- 更新命令行帮助信息
- 添加多个使用示例

## [1.0.4] - 之前版本

### 功能

- 批量下载极客时间专栏文章为 PDF
- 支持 PDF 合并
- 支持并发下载
- 完整的文章内容提取
- 自动添加 PDF 书签
